# Uchiha Elite

Uchiha Clan used to be a fearsome clan before being wiped out by the missing nin Uchiha Itachi. Their ability to copy and counter ninjutsu is unrivalled due to their fantastic doujutsu, the Sharingan.

## Requirements

To qualify to become a uchiha elite, a character must fulfill all the following criteria.
*   **Skills**:  Genjutsu 6 ranks, Knowledge (Tactics or Ninja Lore) 3 ranks, Ninjutsu 6 ranks, Spot 3 Ranks.
*   **Feats**:  Advanced Bloodline (Sharingan Eye).
*   **Special**:  Must be a member of the Uchiha Clan.

## Class Information

The following information pertains to the Uchiha Elite prestige class.

*   **Hit Die**: The Uchiha Elite gains 1d8 hit points per level. The Constitution modifier applies.
*   **Action Points**: The Uchiha Elite gains a number of action points equal to 6 + one-half his character level, rounded down, every time he attains a new level in this class.
*   **Class Skills**: The Uchiha Elite's class skills are as follows. Balance (Dex), Chakra Control (Wis), Concentration (Con), Genjutsu (Cha), Hide (Dex), Jump (Str), Knowledge (Ninja Lore) (Int), Listen (Wis), Move Silently (Dex), Ninjutsu (Int), Read Language (None), Search (Int), Sleight of Hand (Dex), Speak Language (None), Spot (Wis), Taijutsu (Str), Tumble (Dex).
*   **Skill Points at Each Level**: 5 + Int modifier.

## Class Features

The following features pertain to the Uchiha Elite prestige class.

### Doujutsu Expertise

Choose one of the following Masteries. A Mastery cannot be chosen twice unless specified: Doujutsu Proficiency: The Uchiha Elite reduces the chance a creature currently adverting its eyes to avoid making a saving throw by 10% (40% chance not to make a saving throw), but this ability only works if the Sharingan is currently active. This ability can be taken twice, reducing the chance by further 10% (30% chance not to make a saving throw). Improved Technique Copy (Requires Ninja’s Mimicry): The Uchiha Elite is able to use the Memorize function of his Ninja’s Mimicry as an instant action, by spending an action point. He doesn't need to ready an action for that. The user still needs to succeed the perform check and pay half the chakra cost. Sharingan Expertise (Requires 1 other ability from the list): While using See Chakra, the user is able to use Ninja’s Mimicry even if the jutsu does not have hand seals. Doujutsu Counter: Once per day as an instant action, the Uchiha Elite is able to reroll a failed saving throw against any Doujutsu Genjutsu, as long as his Sharingan is currently active. Chakra Colors: If the character concentrates to See Chakra and gain High Speed sight, the duration of the effect is increased by a number of rounds equal to Uchiha Elite's level.

### Bonus Feat

Choose a feat from among the following, for which you must meet any prerequisites: Armor Proficiency, Genin, Chuunin, Genjutsu Adept, Jounin, Mangekyou Sharingan, Ninjutsu Adept, Ninja’s Mimicry, Technique Focus (Uchiha Hijutsu only).
